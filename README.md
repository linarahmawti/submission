# 🚲 Bike Sharing Dashboard ✨

## Deskripsi Project

Project ini merupakan analisis data **Bike Sharing Dataset** untuk mengetahui pola peminjaman sepeda berdasarkan musim, jenis hari, dan tren penggunaan selama tahun **2011 - 2012**.

Dashboard dibuat menggunakan **Streamlit** sebagai media visualisasi interaktif.

---

## Setup Environment - Anaconda
```
conda create --name main-ds python=3.9
conda activate main-ds
pip install -r requirements.txt
```

## Setup Environment - Shell/Terminal
```
mkdir submission
cd submission
pip install -r requirements.txt
```

## Run Streamlit App
```
streamlit run dashboard/dashboard.py
```

### Jika menggunakan macOS
```
python3 -m streamlit run dashboard/dashboard.py
```
